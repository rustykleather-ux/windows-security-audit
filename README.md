# Windows Security Audit

A beginner-friendly Python tool that checks basic Windows security settings and exports results to CSV.

## Checks Planned

- Windows Firewall status
- Windows Defender status
- Local administrator accounts
- BitLocker status
- Password policy
- Recent failed logins

## Why This Project Matters

Security engineers often need to quickly assess endpoint security posture. This project demonstrates practical security automation, Windows administration knowledge, and reporting skills.
